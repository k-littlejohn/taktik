# Taktik

Taktik is a mobile trading simulation app designed to help users backtest forex strategies using real market data and review performance outcomes through a clean, user-friendly interface.

The project focuses on combining software engineering, database design, UI design, and data-driven decision-making into one practical application. Taktik allows users to test trading strategy logic, analyze results, and better understand how different strategies perform across historical market conditions.

## Project Overview

Taktik was built as a forex strategy simulation and backtesting tool. The app uses real market candle data to simulate trades, track results, and display performance statistics in a way that is easier for users to understand.

The goal of the project is not to provide financial advice or execute live trades, but to create a structured environment where users can test strategy ideas and learn from historical performance.

## Key Features

- Forex strategy simulation using real market data
- Backtesting for trading strategies
- Performance tracking for wins, losses, profit, and loss
- User-friendly mobile interface
- Strategy-based workflows
- Historical trade result review
- Data-focused charts and summaries
- Clean app navigation and structured UI flow
